# 产品定位与差异点（赛博神算子）

## 1) 我们的定位是什么？

**一句话定位**：把“线下塔罗/解梦的仪式感”搬到线上，用赛博朋克叙事包装成**每日可复用的心理镜像工具**，让用户用更低门槛得到“可执行的提醒与行动建议”。

**核心人群**（今晚 MVP 先抓住的）：
- 爱塔罗/解梦/玄学内容，但不想被硬推销、也不想被冗长话术轰炸的人
- 情绪波动大、需要“一个能快速帮我定一下今天怎么做”的轻量用户
- 喜欢“风格化输出”（犀利、幽默、赛博）的内容消费者

## 2) 它与线下塔罗的差别是什么？

- **线下塔罗的优势**：高信任关系、强互动、读牌师经验与临场追问
- **我们能做的优势**：
  - **更频繁**：每天/随时可用，成本低到可以“日常化”
  - **更结构化**：固定结构输出，避免“说了很多但没落地”
  - **更可积累**：每次结果可保存成卡片/笔记，形成个人时间线（梦境尤其适配）

## 3) 我们与竞品的差异点是什么？

竞品常见两类：
- 内容农场式“塔罗抽卡/运势”：模板化、泛化、缺少个体语境
- 纯聊天式 AI“陪聊玄学”：能聊但很散，输出不可复用、不可归档

我们的差异点（可做成护城河）：
- **“牌面 → 结构化结论 → 可执行建议”** 的固定格式
- **梦境 × 塔罗** 的双体系融合（把梦境当长期数据资产）
- 强风格：赛博朋克 + 犀利幽默（降低尴尬感，提升传播性）

## 4) 我们的价值怎么体现？

用一句更硬的价值表达：
- **不是预测未来**，而是给你一个“更好做决定”的镜子：让你从混乱里拿到 2~3 个可执行动作。

价值落地的关键指标（后续迭代用）：
- 用户保存/分享率（卡片保存、分享）
- 复访率（每日签连续打开）
- 梦境记录条数（如果做时间线功能）
